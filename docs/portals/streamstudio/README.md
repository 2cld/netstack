# Blender
## Blender Video Edit
- Blender GPU enable
  - Preferences -> System -> Cycles -> Cuda
  - Use defaults, but Video Sequencer
    - Memory Cache Limit 6000
    - Disk Cache Checked
    - Cache Limit 100
    - Compression None
    - Proxy Automatic
- [Blender Video Editing Tutorial](https://www.youtube.com/playlist?list=PLalVdRk2RC6qo7oHp5OO8e7RMe46nYdOY)
