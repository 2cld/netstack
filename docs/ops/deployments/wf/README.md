## Overview

Storage and services site. Runs Mikrotik router, Synology NAS (Docker host), and Proxmox hypervisor. Connected via Starlink ISP and ZeroTier overlay VPN.

## Services (via Cloudflare Tunnel)

- sg.klopfenstein.org
- gitea.klopfenstein.org
- metube.klopfenstein.org
- jp.klopfenstein.org
